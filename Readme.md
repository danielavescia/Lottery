Projeto: SISTEMA DE APOSTAS
Descrição: O projeto consiste no desenvolvimento de um sistema de controle de apostas

O objetivo é permitir que um usuário possa  registrar apostas, registrar apostas com sorteio de números aleatórios.
Requisitos
versão: .net 6.0 ou superior)

Uso
1 - Abra o terminal de comando 2 - Navegue até diretório do projeto da webapi 3 - Execute o comando :dotnet run
1 - Abra outro terminal de comando 2 - Navegue até diretório do projeto do reactapp 3 - Execute o comando :npm start

OU

Compile o programa na IDE e clique em RUN;
